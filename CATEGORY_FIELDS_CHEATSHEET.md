# Feed Post Category Fields Cheat Sheet

Quick reference for what fields to fill in for each category type.

---

## 📝 Common Fields (All Categories)

Every feed post has these base fields:
- ✅ **Title** - Main heading
- ✅ **Slug** - URL-friendly version
- ✅ **Category** - Select from dropdown
- ✅ **Description** - Short summary (1-2 sentences, max 200 chars)
- ✅ **Featured Image** - Main image (optional for some categories)
- ✅ **Body** - Rich text content
- ✅ **Published At** - Date/time
- ✅ **Featured** - Checkbox for homepage feature

---

## 🤖 Prompt of the Week

**Required:**
- Title (e.g., "Best Prompt for Vintage Poster Art")
- Category: `prompt-week`
- Prompt Text (the actual AI prompt)
- AI Tool (e.g., "Midjourney v6", "ChatGPT-4")

**Optional:**
- Prompt Result (image of the AI output)
- Description (quick summary)
- Body (explain why it works, tips, variations)

**Example:**
```
Title: "Cinematic Portrait Lighting Prompt"
Prompt Text: "professional studio portrait, dramatic lighting..."
AI Tool: "Midjourney v6"
Prompt Result: [Upload AI-generated image]
Body: Why this works: The key is in specifying "dramatic lighting"...
```

## 📖 Chicken Chronicles

**Required:**
- Title (story title)
- Category: `chronicles`
- Character Name (protagonist)
- Episode Number (1, 2, 3...)
- Body (the story, 200-300 words)

**Optional:**
- Story Illustration (main artwork)
- Description (story teaser)

**Example:**
```
Title: "The Great Egg Heist"
Character Name: "Captain Cluck"
Episode Number: 3
Story Illustration: [Upload character illustration]
Description: "Captain Cluck hatches a plan to steal back the golden egg..."
Body: [Full story text]
```

## 🛠️ Tool Tuesday

**Required:**
- Title (catchy review title)
- Category: `tool-tuesday`
- Tool Name (official name)
- Featured Image (tool logo/screenshot)

**Recommended:**
- Tool Website (affiliate link if available)
- Tool Rating (1-5 stars)
- Tool Price (e.g., "Free", "$19/mo")
- Tool Pros (bullet points)
- Tool Cons (bullet points)
- Body (your personal experience)

**Example:**
```
Title: "Notion: The Tool That Runs My Business"
Tool Name: "Notion"
Tool Website: "https://notion.so?ref=yourlink"
Tool Rating: 4.5
Tool Price: "$10/month"
Tool Pros: ["All-in-one workspace", "Great templates", "Affordable"]
Tool Cons: ["Steep learning curve", "Can be slow"]
Body: I've been using Notion for 2 years and here's what I've learned...
```

## 📈 Solopreneur Sundays

**Required:**
- Title (e.g., "Week 12: Breaking $1K")
- Category: `solopreneur`
- Week Number (your journey week)
- Revenue (this week's $)

**Recommended:**
- Metrics (array of objects):
  - Label: "Visitors", Value: "1,234", Change: "+15%"
  - Label: "Conversions", Value: "23", Change: "+8%"
- Wins (array of achievements)
- Losses (array of challenges)
- Body (detailed insights, lessons learned)

**Example:**
```
Title: "Week 12: First $1,000 Month"
Week Number: 12
Revenue: 1247
Metrics:
  - { label: "Visitors", value: "2,341", change: "+45%" }
  - { label: "Sales", value: "18", change: "+125%" }
Wins: ["First 4-figure month", "Featured on Product Hunt", "10 new customers"]
Losses: ["Burned out from overwork", "Lost a client"]
Body: What a week. Here's what happened...
```

## ☕ Sunday Swings

**Required:**
- Title (essay title)
- Category: `sunday-swings`
- Body (300-500 word essay)

**Optional:**
- Featured Image (hero image)
- Description (pull quote or summary)

**Example:**
```
Title: "On Staying Small in a Scale-Obsessed World"
Description: "Why I'm building a $10K/mo business instead of a $10M one"
Featured Image: [Upload contemplative photo]
Body: Everyone talks about scaling. But what if you don't want to scale?...
```

## 💡 Quick Tips

### Images
- **Prompt Week:** Result image should be high quality
- **Chronicles:** Illustration should be character-focused
- **Tool Tuesday:** Use tool logo or clean screenshot
- **Solopreneur:** Optional, can use charts/graphs
- **Sunday Swings:** Atmospheric/mood image works best

### Word Counts
- **Prompt:** Explanation 100-200 words
- **Chronicles:** Story 200-300 words
- **Tool Tuesday:** Review 300-500 words
- **Solopreneur:** Update 200-400 words
- **Sunday Swings:** Essay 300-500 words

### SEO
- Title: Make it compelling and searchable
- Description: Front-load keywords
- Slug: Keep it short and descriptive

### Engagement
- **Prompt:** Make prompt copyable
- **Chronicles:** End with cliffhanger
- **Tool:** Include affiliate disclaimer
- **Solopreneur:** Show real numbers
- **Swings:** Ask thought-provoking questions

## 🚀 AI-Assisted Workflow

### Using ChatGPT to Draft Content

**Prompt of the Week:**
```
Create a prompt breakdown for [TOPIC]. Include: the prompt text, 
why it works, and 3 variations to try.
```

**Chicken Chronicles:**
```
Write a 250-word micro-story featuring [CHARACTER] on an adventure 
involving [PLOT]. Make it quirky and family-friendly.
```

**Tool Tuesday:**
```
Create a tool review outline for [TOOL NAME]. Include: what it does,
pricing, pros/cons structure, and use cases.
```

**Solopreneur Sundays:**
```
Structure a weekly update with these metrics: [DATA]. 
Include sections for wins, losses, and lessons learned.
```

**Sunday Swings:**
```
Develop a 400-word reflective essay on [TOPIC]. 
Make it philosophical but relatable.
```

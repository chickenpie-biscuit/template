# 📝 SEO Quick Reference Card

## ✅ Content Editor Checklist

### Every New Blog Post

- [ ] **Meta Title** - 50-60 characters, keyword at start
- [ ] **Meta Description** - 120-160 characters (GREEN in preview)
- [ ] **Focus Keyword** - 1-3 words, main topic
- [ ] **3-5 Additional Keywords** - Related terms
- [ ] **Alt Text on ALL Images** - 10-125 characters, descriptive
- [ ] **Featured Image** - 1200x630px recommended

### Every New Product

- [ ] **Product Description** - Benefits-focused, 150+ words
- [ ] **Meta Description** - Include price, features, CTA
- [ ] **Focus Keyword** - Product type + modifier (e.g., "organic cotton t-shirt")
- [ ] **Multiple Product Images** - All with alt text
- [ ] **Custom OG Image** - For social sharing (optional but recommended)
- [ ] **Customer Reviews** - Add if available

### Every Feed Post

- [ ] **Category Selected** - Determines layout and schema
- [ ] **Featured Image** - Category-appropriate
- [ ] **Alt Text** - On featured image
- [ ] **Meta Description** - 120-160 characters
- [ ] **Keywords** - 2-4 relevant to category

## ✏️ Writing Alt Text

### Formula
`[Subject] + [Action/Context] + [Setting/Style]`

### Examples

**Product Image:**
```
✅ White organic cotton t-shirt on minimalist background
❌ T-shirt
❌ Image of a shirt
```

**Blog Header:**
```
✅ Designer working on laptop with coffee in modern studio
❌ Person at desk
❌ Workspace
```

**Infographic:**
```
✅ Step-by-step guide showing website SEO optimization process
❌ Infographic
❌ SEO steps
```

## 🔑 Keyword Research Tips

### Where to Find Keywords

1. **Google Search** - Check "People also ask" and related searches
2. **Competitor Pages** - What keywords are they targeting?
3. **Your Analytics** - What terms are people already using?
4. **Customer Language** - How do they describe your products?

### Good vs Bad Keywords

**Good (Specific, Intent-Clear):**
- "sustainable organic cotton t-shirts"
- "eco-friendly graphic design services"
- "Filipino chicken adobo recipe easy"

**Bad (Too Generic):**
- "t-shirts"
- "design"
- "food"

## 🚦 Meta Description Color Codes

**🟢 GREEN (120-160 chars)**
- Perfect length
- Will display fully
- Optimal for SEO

**🟡 YELLOW (< 120 chars)**
- Too short
- Add more detail
- Google may generate its own

**🔴 RED (> 160 chars)**
- Too long
- Will be truncated with "..."
- Shorten to key points

## 💬 Meta Description Templates

### Product Pages
```
Shop [Product Name] - [Key Feature]. [Benefit]. 
[Material/Quality]. [Price/Offer]. [CTA].
```

**Example:**
```
Shop Organic Cotton T-Shirts - Ultra-soft & sustainable. 
Made from 100% certified organic cotton. Starting at $29. 
Free shipping over $50.
```

### Blog Posts
```
[Learn/Discover] [Topic]. [Key Insight]. [Benefit]. 
[Time/Length]. [CTA].
```

**Example:**
```
Learn GSAP animation techniques for stunning websites. 
Step-by-step tutorial with code examples. Perfect for 
intermediate developers. 15-minute read.
```

### Category Pages
```
Browse [X] [Category Items]. [Feature 1], [Feature 2], and 
[Feature 3]. [Unique Value]. [Shipping/Return Policy].
```

**Example:**
```
Browse 50+ sustainable fashion pieces. Organic materials, 
ethical production, and modern designs. Carbon-neutral 
shipping. 30-day returns.
```

## ⭐ Adding Product Reviews

### Review Requirements

- **Author Name:** First name or "Verified Customer"
- **Rating:** 1-5 stars (be honest!)
- **Review Text:** 20-500 characters minimum
- **Date:** Automatically added
- **Verified Purchase:** Check if real customer

### What Makes a Good Review

**Includes:**
- Specific details about product
- Use case or occasion
- Fit/quality/experience details
- Would they recommend it?

**Example:**
```
★★★★★ 5/5
"Perfect summer tee"

Love the softness of this organic cotton! Fits true to size, 
washes well, and the cut is flattering. Wore it on a road 
trip and it didn't wrinkle. Highly recommend for anyone 
looking for sustainable basics.

- Sarah M., Verified Purchase
```

## 🎨 OG Image Best Practices

### Design Tips

1. **Text:** Large, readable fonts (60px+ minimum)
2. **Contrast:** Dark text on light background or vice versa
3. **Branding:** Include logo but don't dominate
4. **Focal Point:** Main element centered
5. **Border:** 40-60px safe area from edges

### Tools

- **Canva** - Easy templates
- **Figma** - For designers
- **RemoveBG** - Remove backgrounds
- **TinyPNG** - Compress images

## 🔍 Testing Your SEO

### Before Publishing

1. **Preview Meta Description** - Is it GREEN?
2. **Check All Images** - Do they have alt text?
3. **Review Keywords** - 3-5 filled in?
4. **Read Title Aloud** - Does it make sense?

### After Publishing

1. **Check Live Page** - Looks good?
2. **Share on Social** - Image showing correctly?
3. **Google Search** - Type: `site:yoururl.com`
4. **Rich Results Test** - Validate structured data

## 🆘 Quick Fixes

### "My meta description is red!"
→ Remove fluff words, get to key points faster

### "I can't think of keywords!"
→ Look at your page title, what would you search for?

### "Alt text validation failing!"
→ Must be 10+ characters, add more detail

### "Preview not showing?"
→ Save draft first, preview updates automatically

## 💡 Pro Tips

1. **Front-load keywords** - Put most important words first
2. **Write for humans first** - SEO second
3. **Use active voice** - "Shop now" not "Can be shopped"
4. **Include numbers** - "5 tips" performs better than "Tips"
5. **Ask questions** - "Looking for sustainable fashion?"
6. **Urgency words** - "Limited", "New", "Exclusive"
7. **Review competitors** - See what ranks, do better
